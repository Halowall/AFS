## 1. Aufgabe

严格地说，长度为 0 的 Wort 只有一个：`λ`，也可写作 `ε`。

所以不能给出两个不同的长度 0 的词。`λ` 和 `ε` 只是同一个 leeres Wort 的两种记法。

## 2. Aufgabe

`λ` 是 leeres Wort，是一个 Wort，长度为 0。

`∅` 是 leere Menge，是一个集合，不含任何元素。

区别：

- `λ ∈ A*`
- `∅ ⊆ A*`
- `{λ}` 是只含空词的语言
- `∅` 是不含任何词的语言

共同点：二者都表达某种“空”，但属于不同类型。

## 3. Aufgabe

Wort 被定义为长度为 `n` 的 Zeichenfolge。如果 `n∈N` 且允许 `n=0`，那么长度为 0 的 Zeichenfolge 自然存在。

因此 leeres Wort 是定义的零长度特例。它保证了：

- `A^0={λ}`
- Sternhuelle `A*` 包含长度 0 的词
- 连接运算有单位元：`λw=wλ=w`

## 5. Aufgabe

### a)

若 `L1,L2` 是 `A` 上的形式语言，则 `L1,L2 ⊆ A*`。

因此：

- `L1 ∪ L2 ⊆ A*`
- `L1 ∩ L2 ⊆ A*`
- `L1 \ L2 ⊆ A*`
- `L2 \ L1 ⊆ A*`

所以它们仍然是 `A` 上的形式语言。

### b)

`L1 × L2` 一般不是 `A` 上的形式语言，因为它的元素是有序对 `(u,v)`，不是 Wort。

如果选择编码，例如用新分隔符 `#∉A`：

`encode(u,v)=u#v`

那么可得到语言 `{u#v | u∈L1, v∈L2}`，但这是在扩展字母表 `A∪{#}` 上的语言。

## 6. Aufgabe

构造 TM 计算：

`f(w)=w1`，若 `w` 含奇数个 `1`；否则 `f(w)=w0`。

思路：用状态记录目前看到的 `1` 的奇偶性，扫描到右端 Blank 后写入结果位。

设 Blank 为 `□`：

| 当前 | 读 | 写 | 移动 | 后继 |
|---|---|---|---|---|
| `qE` | `0` | `0` | R | `qE` |
| `qE` | `1` | `1` | R | `qO` |
| `qO` | `0` | `0` | R | `qO` |
| `qO` | `1` | `1` | R | `qE` |
| `qE` | `□` | `0` | N | `h` |
| `qO` | `□` | `1` | N | `h` |

起始状态 `qE` 表示目前 `1` 的个数为偶数。

## 7. Aufgabe

若把第 6 题理解为接受“含奇数个 1 的词”，则 Akzeptor 可这样构造：

1. 用 `qE/qO` 扫描输入，记录 `1` 的奇偶。
2. 扫到右侧 Blank 后，进入返回状态。
3. 一直向左移动到左侧 Blank，再向右移动一格，停在词首。
4. 若最终奇数，进入接受状态；若偶数，进入拒绝状态。

接受语言：

`L={w∈{0,1}* | w 中 1 的个数为奇数}`

## 8. Aufgabe

目标：把输入 `w∈{0,1}*` 当作二进制数，加 1，然后回到第一个非 Blank。

算法：

1. 从左向右扫描到右侧 Blank。
2. 向左执行二进制加一：
   - 读到 `1`：写 `0`，继续向左传递进位。
   - 读到 `0`：写 `1`，进位结束。
   - 读到左侧 Blank：写 `1`，表示最高位进位。
3. 再向左走到 Blank。
4. 向右一格，停机。

例：

- `1011 -> 1100`
- `111 -> 1000`
- `λ -> 1`

## 9. Aufgabe

目标：在 `w∈{0,1}*` 右边再写一次相同的 `w`，得到 `ww`。

可用标记法：

1. 从左到右找第一个未复制的符号。
2. 若是 `0`，标记成 `X`；若是 `1`，标记成 `Y`。
3. 移到当前已写内容的最右端，把对应符号 `0/1` 写到右侧。
4. 回到左端，继续找下一个未复制符号。
5. 所有原始符号都被标记后，将 `X/Y` 恢复为 `0/1`。

这是一种标准 TM 构造。关键是：用标记区分“原词中已复制”和“右侧新副本”。

## 11. Aufgabe

目标：把带内容 `w∈{a,b,c}*` 整体向左移动一格。

1. 先移到输入第一个符号左边的 Blank。
2. 从左到右逐格处理，把右边相邻格的符号复制到当前格。

效果：

原来：

`□ a b c □`

变成：

`a b c □ □`

## 18. Aufgabe

语言：

`L={(ab)^n(ba)^n | n>0}`

构造 KA：

1. 第一阶段读 `(ab)^n`，每读完整一个 `ab` 压入一个 `A`。
2. 非确定地切换到第二阶段。
3. 第二阶段每读完整一个 `ba` 弹出一个 `A`。
4. 输入读完且栈顶为 `k0` 时接受。

核心转移示意：

- `q0` 读 `a` 到 `q1`
- `q1` 读 `b` 压 `A`，回 `q0`
- `q0` 经 `λ` 转到 `q2`，开始读后半部分
- `q2` 读 `b` 到 `q3`
- `q3` 读 `a` 弹 `A`，回 `q2`
- `q2` 在 `λ,k0` 上到接受状态

为了保证 `n>0`，只允许在至少压入一个 `A` 后切换。

## 24. Aufgabe

关系：

`n~m :<=> n-m` 可被 2 整除。

证明：

- 自反：`n-n=0` 可被 2 整除。
- 对称：若 `n-m` 可被 2 整除，则 `m-n=-(n-m)` 也可被 2 整除。
- 传递：若 `n-m` 和 `m-l` 都可被 2 整除，则 `(n-m)+(m-l)=n-l` 也可被 2 整除。

所以是等价关系。对任意固定 `k∈N` 同理。

## 25. Aufgabe

定义：

`A1~A2 :<=> L(A1)=L(A2)`

证明：

- 自反：`L(A)=L(A)`
- 对称：若 `L(A1)=L(A2)`，则 `L(A2)=L(A1)`
- 传递：若 `L(A1)=L(A2)` 且 `L(A2)=L(A3)`，则 `L(A1)=L(A3)`

所以这是自动机集合上的等价关系。

## 37. Aufgabe

### a) `VP -> V NP`

这是基本结构：Verb Phrase 由 Verb 和 Noun Phrase 组成。

例：`sah Krokodil`

### b) `VP -> VP NP`

这是递归规则：一个 VP 后面再接一个 NP。

- `VP -> V NP` 提供基本生成。

## 38. Aufgabe

产生式：

`P={S -> A, A -> aA, A -> bB, B -> bA}`

问题：

1. 没有任何产生式能消去最后的变量，因此无法推导出只含终结符的词。
